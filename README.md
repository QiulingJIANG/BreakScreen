# BreakScreen

A fullscreen break reminder that forces you to rest — because you won't do it yourself.

## Features

- Native desktop app powered by Python + pywebview
- Fullscreen break reminder (Esc to escape, but we'll judge you)

## File Structure
.
├── app.py
├── index.html
├── break-bg.jpg
├── requirements.txt
├── README.md
└── LICENSE

## How To Use

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   python app.py
   ```
